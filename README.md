# ToDoList
Progetto tirocinio FINSOFT

Richieste di sviluppo:

Gestione della lista tramite funzioni di aggiunta, cancellazione, modifica e check conclusivo.
Assegnare ID per gli elementi della lista 
Verificare la correttezza delle info inserite
UI minimalista ma funzionale
Realizzare e gestire una multiutenza
Gestione degli utenti (registrazione, login…)
Avere la possibilità di mostrare gli obiettivi agli altri utenti (solo visualizzazione)

Strumenti adottati per la realizzazione
 
XAMPP: insieme di database utile per la gestione dei dati delle toDoList; come mySQL e Apache.
dBeaver: client che adotteremo per sfruttare al meglio le sue potenzialità al fine del lavoro
Visual Studio Code: software che adotteremo per sviluppare l’applicativo, realizzando il lato di UI con HTML e JavaScript, mentre le funzionalità dell’applicativo in linguaggio php
GitHub: piattaforma che utilizzeremo convenzionalmente per tener conto della consegna, di modo che tutti possiamo apportare le dovute modifiche di pari passo.
